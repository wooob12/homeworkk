웹사이트에서 Repository를 생성하고 주소 복사하기
먼저 비주얼 코드 스튜디오에서 ctrl + shift + ` 눌러서 터미널 열기
1. git init
2. git remote add origin (Repository 주소) -> 여기서 오류 뜰 경우 git remote rm origin 해주기 3. git add .(띄어쓰기)
4. git commit -m "변경사항"
5. git push origin master
-> 여기서 오류 날 경우, md 파일 같이 만들어서 그럼
git pull origin master 해주고 git push origin master 쳐주기
